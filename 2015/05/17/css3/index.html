Keyframes介绍
Keyframes被称为关键帧，其类似于做动画的帧。在CSS3中其主要以“@keyframes”开头，后面紧跟着是动画名称加上一对大括号“{…}”，括号中就是一些不同时间段样式规则
例子：@keyframes changecolor{
0%{
background: red;
}
100%{
background: green;
}
}
在一个“@keyframes”中的样式规则可以由多个百分比构成的，如在“0%”到“100%”之间创建更多个百分比，分别给每个百分比中给需要有动画效果的元素加上不同的样式，从而达到一种在不断变化的效果。
经验与技巧：在@keyframes中定义动画名称时，其中0%和100%还可以使用关键词from和to来代表，其中0%对应的是from，100%对应的是to。
Animation属性
Animation主要用来定义过渡动画效果
animation-name:changecolor; //定义动画的名字，必须与@keyframes changecolor一摸一样，否则动画无效，数值可以是none，无动画
animation-duration: 10s; //必须写，动画过渡的时间，这里面时间可以长些
animation-timing-function: ease; //动画的过渡速度
animation-delay: 1s; //动画启动时间 没有数值，默认为0，加载后执行动画
animation-iteration-count:infinite; //动画循环次数，可写具体数值与无限循环（infinite）
可以缩写为：animation: changecolor 5s ease-out 0.2s infinite alternate;
